## 概要
本リポジトリは、Next.js と TypeScript を用いて開発したカレンダーアプリケーションです。
週表示・月表示の切り替え機能を実装し、日付ごとの予定の追加・編集・削除（CRUD）を行うことができます。

## 使用技術
- Next.js 16.1.6
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- react-modal
- date-fns

## 機能一覧
- 週表示・月表示の切替
- 前週／翌週・前月／翌月への移動
- 日付ごとの予定管理（作成・編集・削除）
